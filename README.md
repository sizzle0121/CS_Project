# Play Atari Tennis with Deep Reinforcement Learning
## Introduction:  
An AI agent  which is able to play Atari-2600 Tennis and surpass human-player  
## Build Requirement:  
Python 3.5  
Pytorch 0.4.1  
CUDA version is depend on GPU  
__Simply Run it with__ `python main.py`
